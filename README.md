# Group Project Mobile Robotics

Worked on multiple projects involving a mobile robot (iRobot Roomba) using C++ and [Robot Operating System (ROS)](https://ros.org/) as part of the "Group Project Mobile Robotics" course at the University of Bonn during the Winter Semester 2022/2023.

### Unit 01
- Developed basic functions for a mobile robot such as move and stop, SMPA (Sense Model Plan Act) architecture, Adaptive Monte Carlo Localization (AMCL), and point navigation.

### Unit 02
- Implemented a wall follower algorithm integrated with SLAM (Simultaneous Localization and Mapping).

### Unit 03
- Implemented the A*-search algorithm.

### Unit 04
- Explored how to identify objects with a camera on a mobile robot using OpenCV (e.g. identifying blue objects).


Additionally, I worked with tools like RViz and Gazebo.

Another project within this course focused on [Person Detection](https://github.com/namazia0/Person-Detection).
